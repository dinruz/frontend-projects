## Overview

![Status: Completed](https://img.shields.io/badge/Status-Completed-brightgreen?style=for-the-badge)

This repository serves as a central hub for my frontend projects. It features solutions from **The Odin Project** curriculum and challenges from the **Frontend Mentor** platform. My work focuses on semantic HTML, advanced CSS (Flexbox/Grid), interactive JavaScript and high-precision design implementation.

* 📅 **Timeline:** March – Oct 2025

## How to Run

1. Clone the repository:
   `git clone https://github.com/dinruz/frontend-projects.git`
2. Navigate to any project folder.
3. Open `index.html` in your browser or use **Live Server** in VS Code.
