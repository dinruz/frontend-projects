## Overview

This is a solution for Social Links Profile challenge on [Frontend Mentor](https://www.frontendmentor.io/challenges/social-links-profile-UG32l9m6dQ). This solution aims to replicate the provided design and fulfill the challenge instructions.

🗓️ July 2025

 🎯 **Main Goals**: 
 
 Users should be able to:

  ✔ See hover and focus states for all interactive elements on the page. 

  ✔ Focus on writing semantic HTML and using the correct elements based on the content.

  ✔ Ensure visitors can navigate the links only using their keyboard.

  ✔ Train your eye for detail by getting your solution to look similar to the design.
